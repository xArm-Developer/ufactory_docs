# 18.xarm\_set\_jerk

192.168.1.66:18333/ws

设置关节加加速度和Tcp加加速度接口：\
设置关节加加速度和Tcp加加速度。

### Message: <a href="#message" id="message"></a>

```1c
#设置关节加加速度为12979，Tcp加加速度为44445
{
    "cmd": "xarm_set_jerk",
    "data": {
        'jointJerk': 12979,
        'tcpJerk': 44445
    },
    "id": "1"
}
```

\


### Request： <a href="#request" id="request"></a>

**请求类型：Application/json**

* cmd
  * string，必填，请求接口
* date
  * json，必填，请求数据
* id
  * string，必填，请求序列号

\
